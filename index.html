Frontend

  • • Next.js 14 with App Router
  • • Tailwind CSS styling
  • • Zustand state management
  • • Smart search with debounce
  • • Protected admin routes

Backend

  • • Express.js REST API
  • • PostgreSQL with Knex.js
  • • JWT authentication
  • • Fuzzy search with ILIKE
  • • Order management

Shared

  • • Common TypeScript types
  • • Shared utilities
  • • Validation schemas
  • • API response types
  • • Constants & enums
